The Foundation.

The Citadel treats complex objectives as deterministic pipelines. Every task is a Bead, every workflow is a Molecule, and every mission can scale across autonomous agents.

Beads: the atomic ledger

Each task is a Bead with strict state transitions: open → in_progress → verify → done.

Agents: specialized workforce

Router, Worker, and Evaluator roles keep planning, execution, and verification distinct.

Dynamic bonding

Workers can spawn new Beads to fan out work while staying accountable.

Project awareness

AGENTS.md rules merge automatically to keep outputs aligned with local standards.

Data piping

Chain tasks dynamically by piping structured output from one Bead to the next.

Schema contracts

Formulas are the autonomy control plane.

Formulas define repeatable processes in TOML. Add variables, dependencies, loops, and recovery logic. The Router compiles them into executable Molecules.
