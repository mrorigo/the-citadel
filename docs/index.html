Principles that turn chaos into craft

State > Compute

Durable beads outlive any worker. Every decision is anchored in the ledger, not in ephemeral memory.

Spec-driven output

Humans define the schema. Agents deliver only implementation — no architecture drift, no surprises.

Nondeterministic, idempotent

Each attempt can differ, but the artifact converges to the same verified result.

Continuous integration as law

No merge without passing gates. The refinery rejects anything that fails tests, lint, or spec.

A compute plane built like a factory

The Router

Translates human intent into atomic beads and queues, controlling the task DAG.

The Worker

Stateless executors consume exactly one bead, producing a diff and exiting cleanly.

The Supervisor

Enforces heartbeats, retries, and cost limits. Workers are cattle, not pets.

The Gatekeeper

Serializes merges, enforces stacked diffs, and keeps the refinery green.
